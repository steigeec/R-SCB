# R-SCB
2 hour workshop for UC Berkeley Society for Conservation Biology

## Topics

1. The data

2. Variable assignment

3. Data types

4. Data structures

5. Data visualization with ggplot2
- Histograms  
- Boxplots  
- Scatterplots  
  - Facetting
  - Subplotting
- PCA example
- Clustering example

6. Maps
- ggplot2  
- simple features  
- leaflet
